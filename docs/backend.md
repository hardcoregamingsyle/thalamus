# Backend (Convex)

## What is Convex?

Convex is a serverless backend platform that provides: real-time database, serverless functions (queries/mutations/actions), file storage, scheduled functions, and HTTP routes. All backend code lives in `src/convex/`.

Key concepts:
- **Queries** — Read-only, automatically subscribe UI to changes
- **Mutations** — Read-write transactions, deterministic (no network calls)
- **Actions** — Can do anything (API calls, network), but can't read/write DB directly (must call internal mutations/queries)
- **Internal functions** — Only callable from other backend functions, not from client
- **Scheduled functions** — Async background jobs triggered by mutations

## Credit System (AgentBucks)

- Each user has `agentBucksBalance` on their user document
- Every AI call deducts tokens × model price from their balance
- Credits reset daily via cron at 18:30 UTC (midnight IST)
- Admin can configure pricing per model in the `/admin` panel
- Platform-wide spending tracked via `platformBudget` table

## HTTP Routes (src/convex/http.ts)

| Method | Path | Purpose |
|--------|------|---------|
| POST | `/stream-chat` | SSE streaming for chat/study modes |
| OPTIONS | `/stream-chat` | CORS preflight |
| POST | `/github/webhook` | GitHub push events for auto-sync |
| GET | `/github/callback` | OAuth code exchange → token storage |

The streaming endpoint (`/stream-chat`) implements full SigV4 signing for AWS Bedrock and parses the binary event-stream protocol to deliver real-time token-by-token responses via Server-Sent Events.

## Cron Jobs

Single cron: **Reset daily agent bucks** at `30 18 * * *` (18:30 UTC = 00:00 IST)
- Calls `internal.dailyReset.resetDailyAgentBucks`
- Resets every user's `dailyAgentBucks` to 10,000,000
